# HF🤗每日简报机器人

该机器人能自动从 [🤗 Daily Papers](https://huggingface.co/papers) 收集paper信息，然后翻译成已经设定的语言

# 🤗每日简报机器人如何工作？

1、获取 [🤗 Daily Papers](https://huggingface.co/papers) 中的所有paper信息

    **包括**：

        ①、论文标题

        ②、arxiv ID

        ③、论文摘要


      

2、将收集到的paper信息翻译成已经设定的语言

# 🤗每日简报机器人未来准备做？

- [ ] 自动翻译成更多的语言 (第一目标 -> 简体中文)
- [ ] 自动翻译成更多的语言 (第二目标 -> 繁體中文)
