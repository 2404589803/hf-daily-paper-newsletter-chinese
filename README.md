# HF🤗每日简报机器人

该机器人向订阅者发送从 [🤗 Daily Papers](https://huggingface.co/papers) 收集的paper信息

# 🤗每日简报机器人如何工作？

1、获取 [🤗 Daily Papers](https://huggingface.co/papers) 中的所有paper信息

2、将paper信息翻译成中文

3、将paper信息发送到订阅者的邮箱
